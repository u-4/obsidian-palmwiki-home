# 既知の制限

- 0.2.0はObsidian Desktop 1.12.7以上を対象とし、macOS版1.12.7で確認しています。iOS / Android対応は今後検証します。
- 本文全文検索、OCR検索、複数Vault検索、関連ページスコアは未実装です。
- PageRankはGoogle PageRankそのものではなく、Vault内のリンクと更新日時を利用した静的な重要度指標です。
- PageRankは解決済みのMarkdown間リンクだけを使用します。タグ、未解決リンク、Markdown以外の埋め込み、OCR本文は対象外です。
- PageRankの除外パターンが正規表現として不正な場合は部分一致として扱われ、設定画面にはエラーを表示しません。
- 除外設定はPageRankへの影響だけを止めます。表示上の被リンク数とリンク数は変わりません。
- フォルダー設定はテキスト入力で、フォルダーピッカーはありません。
- タグ絞り込みは完全一致です。
- テーブル行は仮想化の安定性を優先して固定高のため、長い説明や多数のタグは省略表示されます。
- 保存済み索引を先に表示するため、`Waiting`または`Indexing`の間は変更前の情報が一時的に見える場合があります。`Complete`で最新状態への検証が完了します。
- 初回利用時、キャッシュ破損時、対象設定の変更時には索引を作り直すため、Vaultの規模に応じた待ち時間が発生します。
- 表示用索引と本文メタデータはVault内のプラグイン専用キャッシュへ保存されます。画像URLと日付表示の小さなキャッシュはメモリ上だけです。
- 約7,000ページのコピーVaultで表示速度と基本操作を確認していますが、PageRank順位の品質はVaultのリンク構造に左右されます。
- 左上Homeボタンの配置は、Obsidianに公開された専用APIがないため、Obsidianの左側ヘッダーDOM構造へ限定的に依存します。Obsidian側のヘッダー構造が将来変わった場合は、ボタンだけが表示されなくなる可能性があります。
- Obsidianコマンドの一覧取得と実行は、公開された`App`型に含まれないcommand managerを実行時に安全確認して利用します。非対応のObsidian版、無効化されたコマンド、現在のタブで実行できないコマンドでは、画面を変えずNoticeを表示します。
- Obsidian起動直後でまだ読み込まれていない非アクティブな復元タブでは、手入力した相対ページ・ローカル見出し・Markdown文脈依存コマンドが、そのタブの読み込み完了まで利用できない場合があります。ページ選択画面で保存した正確なVault相対パスには影響しません。
